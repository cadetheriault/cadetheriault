# Doodle Dash 
# Made in 2020

## Description

This game was created for the end of year project in my final year of high school (2020).
The goal was to create a game of any kind, using java and the skills we had learned 
throught the school year.

My goal was to learn how to create a full and real-time interactive game with a GUI.

## Installation

java version "1.8.0_311" or later should be installed 
(previous versions may work, but issues may arise).


Java(TM) SE Runtime Environment (build 1.8.0_311-b11)
Java HotSpot(TM) 64-Bit Server VM (build 25.311-b11, mixed m
